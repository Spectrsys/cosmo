
Cosmo Sharing Server 0.2.1
Release Notes
        

UPGRADING FROM A PREVIOUS RELEASE
=================================

Due to incompatible schema changes, 0.2.1 cannot operate against
repositories created by earlier versions of Cosmo.


BUGS FIXED
==========

2854: WebDAV resources with spaces in their names have their URLs
      correctly escaped in PROPFIND responses
3068: Apostrophes are now accepted in WebDAV resource names
4102: The `root' user can no longer be deleted via CMP
4103: CMP ensures that the username in the URI and in the XML request
      content are the same
4104: CMP no longer includes a homedirUrl in the XML representation of
      the `root' user
4212: "Expensive" WebDAV properties are no longer calculated when
      those properties are not needed in the response
4225: The repository is now cleanly shut down when the server is
      stopped.
4226: Ticket validation now properly handles lookups for resources
      with spaces in their names
4255: Duplicate UID queries for CalDAV event resources now properly
      handle resources with spaces in their names
