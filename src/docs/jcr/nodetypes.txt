iCalendar
=========

iCalendar calendars, components, and properties are represented as JCR
nodes.

Property values and parameters are represented as JCR properties and
converted to JCR property types (date, long, double, etc.) where
possible, defaulted to String otherwise.

Complex values are broken down into multiple JCR properties (eg
icalendar:version has two properties, icalendar:minVersion and
icalendar:maxVersion).

Inheritance Hierarchy:

  nt:base
  |
  |--icalendar:calendar
  |
  |--icalendar:component
  |  |
  |  |--icalendar:event
  |  |
  |  |...
  |
  |--icalendar:property
  |  |
  |  |--icalendar:textproperty
  |  |  |
  |  |  |--icalendar:description
  |  |  |
  |  |  |...
  |  |
  |  |--icalendar:datetimeproperty
  |  |  |
  |  |  |--icalendar:dtend
  |  |  |
  |  |  |...
  |  |
  |  |--icalendar:prodid
  |  |
  |  |...
