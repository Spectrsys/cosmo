Provides utilities and implementation classes for integrating the Acegi Security System for Spring into Cosmo.

Package Specification

This package provides two types of classes:

  • Servlet filters that control the instantiation and behavior of various Acegi Security framework components
  • Services that use Cosmo data access objects to adapt the Cosmo model to the Acegi Security model

The ticket and homedir subpackages contain classes that together implement Cosmo's custom authentication and authorization schemes.

Cosmo defers to Acegi Security's role-based security implementation for access control for administrators.
