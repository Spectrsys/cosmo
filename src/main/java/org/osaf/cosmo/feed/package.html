Provides an Atom 1.0 representation of repository collections via HTTP.

Package Specification

This package provides rudimentary support for accessing the contents of Cosmo repository collections via Atom 1.0 feeds.

Eventually this package will be expanded to provide full HTTP-only read-write access to all collections and resources, as a lightweight alternative to CalDAV and other protocols. The most likely directions for Atom integration include GData and/or CalAtom.
